Ubiquity for Opera

Help

You're probably wondering what Ubiquity for Opera is or how does it work. The original Ubiquity is a Firefox extension that brings a textual interface to the browser.
Ubiquity for Opera is an attempt to port this extension to the Opera browser, using the UserJS feature, a sort of built-in GreaseMonkey that Opera has.

Due to UserJS security protection and limitations, some of the features of the original Ubiquity are not available in the Opera version. Particularly, no access to external HTTP resources is allowed, and there's no access to the browser chrome... However, Ubiquity for Opera is still useful. Try it!

How does Ubiquity work?

You can start using it, if you already installed and configured it, by typing: CTRL + SPACE.
This is the default shortcut that brings up the Ubiquity window. This is nothing more than a simple text field.

You just type the command you want to execute, ex.: clusty opera browser and there you go, you get the search results. Here's a screenshot:

Example of commands

Some examples of the commands you can try in Ubiquity:

  • google Opera UserJS tutorial
  • imdb Matrix
  • maps Colosseo, Roma
  • Select a word or sentence on the page you're browsing, and then type translate without any word. It will pick up the selected text and translate it through Google Translate.
  • youtube Paul McCartney
  • dramatic-chipmunk
  • validate, will validate through the W3C Validator service the currently opened window URL

How to install Ubiquity for Opera

If you landed on this page, you probably have Ubiquity already installed. If not, follow these simple steps:

  • Enable UserJS. Point your browser to opera:config#Javascript and:
    • tick the "User Javascript" checkbox
    • in the "User Javascript File" textfield, type the name of the folder you want to run the scripts from. You can create an empty one wherever you want.
    • If you want, you can also enable the checkbox to allow UserJS execution in HTTPS pages.
  • Remove the default shortcut key bound to CTRL + SPACE, since the original Ubiquity is activated and deactivated with CTRL + SPACE.
